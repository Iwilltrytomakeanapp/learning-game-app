# 🎮 LearnQuest - Interactive Learning Game

An engaging, gamified learning platform designed to make Math, Science, and Social Science more fun for kids and advanced learners alike!

## 🎯 Features

- **Multiple Subjects**: Math, Science, Social Science
- **Tiered Difficulty Levels**: Beginner → Intermediate → Advanced → Expert
- **Gamified Learning**: Points, badges, leaderboards, and progress tracking
- **Kid-Friendly Interface**: Colorful, intuitive design with age-appropriate content
- **Interactive Challenges**: Quizzes, puzzles, mini-games, and real-world scenarios
- **Progress Tracking**: Personalized learning paths and achievement system
- **Adaptive Difficulty**: Content adjusts based on performance

## 🚀 Getting Started

### Prerequisites
- Node.js 16+
- MongoDB
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Iwilltrytomakeanapp/learning-game-app.git
cd learning-game-app

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Start the development server
npm run dev
```

## 🎓 How It Works

### Difficulty Tiers

1. **Beginner (Ages 6-8)**
   - Simple concepts with visual aids
   - Multiple choice questions
   - Encouraging feedback

2. **Intermediate (Ages 9-11)**
   - More complex problems
   - Mixed question types
   - Challenge badges

3. **Advanced (Ages 12-14)**
   - In-depth problem-solving
   - Real-world applications
   - Competitive elements

4. **Expert (Ages 15+)**
   - Advanced concepts
   - Complex scenarios
   - Leadership board challenges

### Game Mechanics

- **XP System**: Earn experience points for completing challenges
- **Badges**: Unlock special badges for achievements
- **Streaks**: Maintain learning streaks for bonus points
- **Challenges**: Daily and weekly challenges with rewards
- **Leaderboards**: Compete with friends or globally

## 🎨 Design Philosophy

- **Colorful & Engaging**: Bright colors, fun animations, and positive feedback
- **Progressive Challenge**: Content difficulty increases naturally
- **Accessible**: Clear fonts, readable text, colorblind-friendly palette
- **Rewarding**: Frequent achievements and positive reinforcement
- **NOT Duolingo-like**: Unique game mechanics, diverse question types, story-driven learning

## 📊 Progress Tracking

Users can view:
- Overall progress by subject
- Time spent learning
- Accuracy rates
- Badges earned
- Challenges completed
- Comparison with previous performance
